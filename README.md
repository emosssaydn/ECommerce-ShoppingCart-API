# E-Commerce Shopping Cart Calculation Service

This repository contains a robust pricing and discount calculation service designed for scalable modern e-commerce platforms. 

## Key Features & Architectural Decisions

### 1. Applied Concepts
* **SOLID Principles:** Business logic is entirely decoupled from the database or API controller layer.
* **Clean Code & Readability:** Complex, nested if-else statements are replaced with intuitive LINQ `.Sum()` expressions.
* **High Testability:** The service relies exclusively on decoupled standard objects, making it ready for comprehensive automated Unit Tests.

### 2. Tech Stack
* **Language:** C#
* **Framework:** .NET Ecosystem
* **Methodology:** LINQ, Domain-Driven Logic Separation
